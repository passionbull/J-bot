# J-bot

Our purpose is to make affordable assistant robot.

It is currently based on google AIY voice kit.

Later, I wanna make afforable assistant robot depending any linux board.

# List of materials
* Raspberry pi
* Camera
* Speaker & microphone
* Button
* Arduino
* Servo motors
* Ultra sonic sensors

# System

* OS : linux
* Framwork : ROS


# Functions

* Google-assistant
* Vision works
* Sensing and moving (with arduino)
